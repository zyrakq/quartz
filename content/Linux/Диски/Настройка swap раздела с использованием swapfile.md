### Добавляем swap

Требуются права администратора.

```sh
touch /swapfile
```

```sh
 chattr +c /swapfile
```

```sh
fallocate -l 8G /swapfile
```

```sh
chown root /swapfile
```

```sh
chmod 600 /swapfile
```

```sh
mkswap /swapfile
```

```sh
swapon /swapfile
```

```sh
echo '/swapfile none swap defaults 0 0' >> /etc/fstab
```

### Проверяем наличие swap

```sh
swapon --show
```

```sh
free -h
```

Права администратора не требуются.
### Отключаем swap

```sh
swapoff -a &&` `sleep` `3 && swapon -a
```

Права администратора не требуются.
